# Калькулятор бассейна

Веб-приложение для расчета стоимости строительства бассейна.

## Функциональность

- Расчет размеров бассейна
- Расчет объемов земляных работ
- Расчет армирования
- Расчет обратной засыпки
- Расчет стоимости оборудования
- Расчет стоимости материалов
- Расчет стоимости работ
- Расчет общей стоимости проекта

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/pool-calc.git
cd pool-calc
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python app.py
```

4. Откройте в браузере:
```
http://127.0.0.1:5003
```

## Технологии

- Python 3.9+
- Flask
- Bootstrap 5
- jQuery
